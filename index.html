How to Set Up Your Google Sheet

  1. Create a new Google Sheet
  2. Add names in the first column (A)
  3. Click the "Share" button in the top right
  4. Change access to "Anyone with the link"
  5. Make sure the permission is set to "Viewer"
  6. Copy the link and paste it in the input field above

Note: The sheet must be publicly accessible for the raffle wheel to work.

Important: Due to browser security restrictions, the Google Sheets integration may not work. If this happens, please use the manual input option above to enter names directly.

Recommended: For the most reliable experience, use the manual input option above.
